How Does AMP's Data-Binding Work?

While AMP restricts inine JS scripts, it can still do data-binding. Data-binding connects HTML properties like visibility and text-content to JS expressions that use user inputs such as buttons and form fields. The AMP component amp-bind uses data-binding to allow you to customize stateful interactivity on a mobile page by binding element property to JS-like expressions that can reference the page's state. This is also called one-way binding - in Angular and React JS they have two-way binding, where any changes made in this model will update in the template, and any changes in the template will update in the model.
